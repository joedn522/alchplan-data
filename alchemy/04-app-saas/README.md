# App / SaaS 開發

> 狀態：Deep thinking 完成 ✅（見 deep-thinking-app-development-guide.md）
> 優先度：中（需驗證需求，但工程師最大優勢所在）

## 核心點子：語音子彈筆記
語音輸入 → AI 解析 → 自動填入子彈筆記 template → 產生 insight

## 已知要點
- 搭配 OpenClaw 大概 2 天可做出 MVP
- 平台建議：Web App (SaaS) > iOS > Android
- 金流：Stripe（簡單、合規、不會有官司）/ 台灣用綠界
- 策略轉變：skill/workflow 不該是產品，而是行銷工具；付費的是教學+思維+社群
- 市場缺口：現有工具（Day One、Notion AI、龍蝦蝦）都沒有完整解決「語音→結構化筆記」
