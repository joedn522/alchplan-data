# 波波獸品牌 — 內容創作

> 狀態：待 deep thinking
> 優先度：中（從零開始，但長期是所有路線的品牌基底）

## 核心定位
波波獸 — 一個 Google 工程師的生活分享品牌（音樂、電影、書籍、酒類、旅遊）

## 已知要點
- 既有品牌：波波獸（followers 很少，幾乎從零）
- YouTube：聽歌波波獸
- 分享主題：音樂 cover、電影、書籍、酒類、旅遊
- AI 結合：AI 腳本、AI 封面圖卡、voice clone、自動逐字稿
- 聲音是核心差異化資產
