# 自動化內容產線

> 狀態：待 deep thinking
> 優先度：高（這是讓其他所有路線可行的基礎設施）

## 核心概念
每天做事的過程本身就是內容，AI 自動化整個產線：一份素材 → 多管道輸出

## 已知要點
- 流程：日常工作/錄音 → AI 逐字稿 → AI 萃取重點 → 自動產出 Podcast/文章/圖卡
- 一魚多吃：Podcast 錄一次 → 電子報 + X 貼文 + IG 圖卡 + show notes
- Build in public 的自動化版本
- 你最熱愛自動化工作流
