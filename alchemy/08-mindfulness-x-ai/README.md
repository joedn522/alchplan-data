## 核心定位
「我是用 AI 的人，但我也練正念、也慢下來思考」— 科技人的正念生活

## 已知要點
- 大部分 AI 內容講效率，你講正念與慢思考，形成獨特反差
- 擅長的領域：正念、刻意練習、番茄鐘、80/20 法則、拆解書籍
- 常看的書：數字、刻意練習、蛤蟆先生去看心理師
- 說書太多人做了，但「AI 工作流 + 正念 + 好聲音」這個交叉幾乎沒人
- 可能更適合作為 Podcast 的主題之一，而非獨立路線
