## 為什麼是大 topic 而不是「找 Microsoft」

雖然 Microsoft 是 Asher 反覆提及的第一目標，但實際上「跳槽」這件事的範疇更大：
- 目標公司不只一個 — 物色多個 role / org 才能比較
- 準備工作有結構 — LeetCode、System Design、行為面試
- 進度需要可視化 — 哪個目標走到哪一階段、刷了幾題

所以這裡是「跳槽計畫」總目錄。Microsoft 是 [targets.md](targets.md) 中**目前的首要目標**。

## 每週 1 步的籃子

從這幾個籃子裡挑一個（不用每週都做同一種，也不用每週都同一個目標公司）：

- **投履歷** — 認真投 1 個職缺（研究過 team / manager 後）
- **連絡人** — 給 1 位前同事 / 認識的人發訊息
- **看 JD** — 認真讀 1 個 JD，做 1 頁筆記
- **刷 LeetCode** — 至少 1 題（標記到 [leetcode.md](leetcode.md)）
- **System Design** — 至少 1 題或讀 1 個案例
- **模擬面試** — 跟自己 / AI 跑 1 輪行為面試或系統設計

週末沒做就由 Alchemy 在下個 cycle 提醒，但不堆積（不會說「你三週沒做要補三步」— 補不來，也沒意義）。
