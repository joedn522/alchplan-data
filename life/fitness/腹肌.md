## 一、腹肌基礎知識（體脂率 vs 訓練）

### 殘酷的真相：腹肌是「練出來的」也是「吃出來的」

每個人都有腹肌，但能不能看見取決於兩個因素：

| 因素 | 重要性 | 說明 |
|------|--------|------|
| **體脂率** | ★★★★★ | 男性需降到 **10-14%** 才能看到明顯腹肌線條；15-17% 可見輪廓 |
| **肌肉量** | ★★★★☆ | 腹肌肌肉越厚，在稍高體脂下也能看到線條 |
| **基因** | ★★★☆☆ | 決定腹肌形狀（對稱/不對稱）、塊數（4/6/8 塊），無法改變 |

### 關鍵認知

- **體脂 > 18%**：不管練多兇，腹肌都被脂肪蓋住，看不到
- **體脂 12-15%**：腹肌訓練的效果才會明顯呈現
- **無法局部減脂**：狂做仰臥起坐不會讓肚子的脂肪消失，必須整體降脂
- **訓練 + 飲食缺一不可**：訓練讓肌肉更厚更立體，飲食控制讓它露出來

### 腹肌的解剖結構

| 肌群 | 位置 | 功能 | 代表動作 |
|------|------|------|----------|
| **腹直肌** (Rectus Abdominis) | 正面六塊肌 | 脊柱屈曲（捲腹動作） | 捲腹、懸吊舉腿 |
| **腹外斜肌** (External Obliques) | 側腹外層 | 旋轉、側屈 | 腳踏車捲腹、側平板 |
| **腹內斜肌** (Internal Obliques) | 側腹內層 | 旋轉、穩定 | 同上 |
| **腹橫肌** (Transversus Abdominis) | 最深層 | 核心穩定、壓縮腹腔 | 平板支撐、真空收腹 |

**全面訓練 = 四個肌群都要練到。**

## 二、最有效的腹肌動作（根據 EMG 研究排名）

> 以下排名參考 ACE (American Council on Exercise) 的 EMG 研究，以及多篇運動科學文獻。EMG（肌電圖）測量肌肉活化程度，數值越高代表該動作越能刺激目標肌群。

### 腹直肌活化排名（以傳統捲腹為基準 100%）

| 排名 | 動作 | EMG 活化度 | 難度 | 備註 |
|------|------|-----------|------|------|
| 1 | **腳踏車捲腹** (Bicycle Crunch) | ~248% | 中 | 同時高度活化腹直肌 + 腹斜肌 |
| 2 | **Captain's Chair 舉腿** | ~212% | 中高 | 健身房器材，下腹為主 |
| 3 | **健腹輪滾輪** (Ab Wheel Rollout) | ~210% | 高 | 全程離心+向心，極強刺激 |
| 4 | **懸吊舉腿** (Hanging Leg Raise) | ~200% | 高 | 需要握力，下腹王牌動作 |
| 5 | **瑞士球捲腹** (Swiss Ball Crunch) | ~139% | 中 | 不穩定面增加活化 |
| 6 | **反向捲腹** (Reverse Crunch) | ~130% | 低中 | 下腹重點，初學者友善 |
| 7 | **平板支撐** (Plank) | ~100% | 低 | 等長收縮，核心穩定基礎 |
| 8 | **傳統捲腹** (Basic Crunch) | 100% (基準) | 低 | 基礎動作 |

### 腹斜肌活化排名

| 排名 | 動作 | 特點 |
|------|------|------|
| 1 | **腳踏車捲腹** | 旋轉動作高度活化斜肌 |
| 2 | **Captain's Chair 側舉腿** | 加入側向抬腿 |
| 3 | **側平板支撐** (Side Plank) | 等長穩定，對斜肌效果佳 |
| 4 | **俄羅斯轉體** (Russian Twist) | 旋轉負重，進階動作 |

### 動作選擇原則

1. **每次訓練至少包含**：1 個上腹動作 + 1 個下腹動作 + 1 個斜肌動作 + 1 個穩定動作
2. **優先選擇 EMG 排名高的動作**
3. **隨能力提升逐步換成更難的變化式**

## 三、4 週漸進式腹肌課表

> 每週 3-4 次，每次 10-15 分鐘

### 漸進超負荷策略

腹肌和其他肌群一樣需要漸進超負荷才能成長：

| 策略 | 說明 | 範例 |
|------|------|------|
| **增加次數** | 每週多 1-2 下 | 捲腹 12 下 → 15 下 |
| **增加組數** | 從 3 組到 4 組 | 適合已經適應的動作 |
| **增加負重** | 手持啞鈴或使用 Cable | Cable 捲腹加重量 |
| **減少休息** | 45 秒 → 30 秒 | 提升代謝壓力 |
| **換更難的變化式** | 屈膝舉腿 → 直腿舉腿 → Toes to Bar | 最重要的進階方式 |
| **增加離心控制** | 放下的過程數 3 秒 | 增加張力時間 |

**不建議**：只靠做更多下來進階。腹肌訓練超過 25 下/組就變成耐力訓練，對肌肥大效果下降。應優先用負重或更難的動作變化式來進階。

## 四、飲食建議（簡要）

### 要看到腹肌，飲食佔 70%

| 原則 | 具體做法 |
|------|----------|
| **熱量赤字** | 每日少吃 300-500 大卡（別太極端，否則掉肌肉） |
| **高蛋白** | 每公斤體重攝取 1.6-2.2g 蛋白質（保肌減脂的關鍵） |
| **減少精緻碳水** | 白飯減量，改糙米、地瓜、燕麥 |
| **充足蔬菜** | 每餐至少一拳大小的蔬菜，增加飽足感 |
| **足夠水分** | 每天 2-3 公升，幫助代謝 |
| **減少酒精** | 酒精熱量高且促進腹部脂肪堆積 |
| **不要節食** | 極低熱量飲食會降低代謝、流失肌肉，反效果 |

### 簡易每日飲食範本（參考）

| 餐次 | 內容範例 |
|------|----------|
| 早餐 | 3 顆蛋 + 燕麥 + 一份水果 |
| 午餐 | 雞胸肉/魚 + 糙米飯（半碗到一碗）+ 大量蔬菜 |
| 訓練後 | 乳清蛋白 1 份 + 香蕉 |
| 晚餐 | 瘦肉/豆腐 + 地瓜 + 蔬菜 |
| 點心（如需要）| 希臘優格 / 水煮蛋 / 堅果（一小把）|

## 五、常見錯誤

### 1. 只練腹肌，不控飲食
> 每天做 200 下仰臥起坐但繼續大吃，腹肌永遠藏在脂肪下面。

### 2. 動作用甩的，不控制
> 速度太快、靠慣性完成動作 = 腹肌沒真正出力。**慢下來**，感受收縮。

### 3. 只做捲腹
> 只練腹直肌上半段。下腹、斜肌、腹橫肌都要練到，才能有完整線條。

### 4. 每天練腹肌
> 腹肌也是肌肉，需要恢復。每週 3-4 次就夠了，中間至少休息一天。

### 5. 平板支撐撐太久
> 超過 60 秒的平板支撐對肌力成長效益遞減。不如換成更難的變化式或增加負重。

### 6. 忽略呼吸
> 正確呼吸：**出力時吐氣（捲起來時），放下時吸氣**。憋氣會增加腹內壓但降低訓練效果。

### 7. 期待兩週就有腹肌
> 現實時間表：
> - 體脂 20% 以上 → 需要 **3-6 個月** 認真控制飲食 + 訓練
> - 體脂 15-18% → 需要 **1-3 個月**
> - 體脂 12-15% → 認真練 **4-8 週** 就能看到明顯進步

### 8. 拉脖子做捲腹
> 手放在頭後面然後猛拉脖子，這樣練的是頸椎不是腹肌。手輕觸耳側或交叉放胸前。

## 六、訓練原則總結

| 項目 | 建議 |
|------|------|
| 頻率 | 每週 3-4 次 |
| 每次時間 | 10-15 分鐘 |
| 動作數 | 4 個動作/次 |
| 組數 | 每動作 3 組 |
| 次數 | 8-15 下（肌肥大範圍） |
| 休息 | 30-60 秒 |
| 最重要的事 | 控制飲食降體脂 |
| 第二重要的事 | 動作品質 > 次數 |
| 搭配 | 全身重訓（深蹲、硬舉等複合動作也大量使用核心） |
