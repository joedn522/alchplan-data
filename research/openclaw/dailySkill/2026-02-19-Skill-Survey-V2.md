## 🌐 網路社群熱門推薦 (Reddit/GitHub/Tech Blogs)
這些是目前在 OpenClaw/MoltBot 社群中被廣泛討論的熱門技能：

### 1. Claude Code (MCP Integration)
*   **核心功用**：將 Anthropic 的 Claude Code 強大編碼能力整合進 OpenClaw。支援子代理 (sub-agent) 協作處理複雜程式開發、狀態持久化。
*   **使用時機**：當你需要我進行大型專案重構、跨多檔案的程式碼修補或複雜的系統部署時。

### 2. last30days (Synthesized Research)
*   **核心功用**：專門針對過去 30 天內的 Reddit、X (Twitter) 及網路資訊進行打撈與綜合分析。
*   **使用時機**：你想知道「最近大家對某個技術/事件的看法」或需要獲取最即時、尚未進入搜尋引擎權重區的新鮮資訊時。

### 3. reddit-readonly (API-less Scraper)
*   **核心功用**：無需 Reddit API Key 即可抓取熱門貼文、評論區。
*   **使用時機**：建立「每日社群趨勢摘要」機器人，自動將你感興趣的 Subreddit 重點推送到 Telegram。

### 4. McPorter (Advanced MCP Controller)
*   **核心功用**：OpenClaw 官方的高級 MCP 管理工具，支援自動生成工具類型定義、配置 MCP 伺服器、甚至處理熱插拔工具。
*   **使用時機**：當你手上有一個 MCP 伺服器 (Node/Python) 但懶得手動寫 SKILL.md 時，用它可以快速橋接。

### 5. Browser-JS (Enhanced Web Automation)
*   **核心功用**：比內建 browser 更強大的網頁自動化，支援執行複雜 JavaScript、處理動態渲染網頁，甚至自動跳過機器人驗證。
*   **使用時機**：需要抓取像是股票交易網站、動態儀表板等高度動態的網頁內容時。

## 🤖 ClawHub 深度調查 (Top 10-20 精選)

### 6. mac-power-tools (macOS 系統深度整合)
*   **核心功用**：控制 macOS 音量、深色模式、行事曆、甚至執行 AppleScript。
*   **使用時機**：想把 OpenClaw 變成你的 Mac 管家，自動幫你處理系統排程任務。

### 7. composio-integration (600+ Tool Integration)
*   **核心功用**：這是一個「萬能介面」，讓 OpenClaw 能瞬間連接 Slack, Gmail, GitHub, Salesforce 等 600 多個第三方服務。
*   **使用時機**：當你想連接某個還沒有專屬 OpenClaw 技能的商業軟體時。

### 8. cli-developer (CLI 工具開發助理)
*   **核心功用**：輔助開發者編寫、Debug、並自動生成 Shell 指令文件。
*   **使用時機**：我們正在開發自己的 .sh 腳本或管理 NAS 節點時，它可以提供更高階的建議。

### 9. qveris (Query Verification)
*   **核心功用**：在 AI 執行重要指令前進行安全檢索與驗證。
*   **使用時機**：對安全性要求極高時（如刪除檔案、更動資料庫前的一道保險）。

### 10. meow-finder (Smart File Search)
*   **核心功用**：針對本地工作區檔案內容進行語義級別的快速檢索。
*   **使用時機**：專案檔案很多，你想問「我上次在哪個檔案寫過關於快取邏輯的部分」。
