# 30 - Antigravity 工作 SOP 設計

目標：讓 Antigravity 在常見情境中「一步就做對」，而不是每次都 ad-hoc 聊天。

## 想像中的使用方式
- 給定一個固定的入口指令（例如在 Chat / CLI / 工具欄）
- 依照情境自動幫你：
  - 問對問題（補 context）
  - 選對工具（搜尋、code、doc、dashboard 等）
  - 回傳一份接近可直接執行的結果

## 待整理主題
- [ ] Antigravity 目前支援的能力（code, search, tools, workflow？）
- [ ] 可以被「封裝成一鍵 SOP」的情境列表
- [ ] Prompt / preset 設計：怎麼把「我的工作習慣」固化進去
- [ ] 與公司內其他工具（bug tracker / code review / dashboard）的串接可能性
