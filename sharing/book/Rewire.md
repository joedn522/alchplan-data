## 全書地圖

### 前言：神經可塑性概論
大腦的可塑性基礎，以及為什麼 90% 的日常決策都是潛意識自動運作的。

### 第一階段：擺脫負面情緒
| 主題 | 核心概念 |
|---|---|
| 識別觸發因素 | 大腦天生偏負面（負面偏誤），先認識這個原廠設定 |
| 生理性嘆息 | 雙吸氣→長吐氣，啟動副交感神經，快速降壓 |
| 戰略性休息 | 壓力和焦慮不是同一件事；用策略性暫停打斷慢性壓力 |
| 視覺化想像 | 想像啟動和真實行動相同的神經迴路 |
| 重新架構想法 | 語言塑造思考，思考塑造感受，感受決定行為 |
| 言語的力量 | 自我對話的品質直接影響神經迴路的強化方向 |

### 第二階段：改變你的敘事
| 主題 | 核心概念 |
|---|---|
| 放下手機 | 手機劫持多巴胺迴路，讓大腦難以重組 |
| 心理訓練 | 刻意練習新的思維模式，像訓練肌肉一樣 |
| 重複的力量 | 21天打斷舊迴路，持續重複才能建新高速公路 |
| 暫停（留白）| 在刺激和反應之間創造空間，是改變的關鍵時刻 |
| 突破瓶頸 | 舒適區的邊界就是大腦重組最活躍的地方 |
| 準備應對挫折 | 提前設計「如果⋯那就⋯」的應對策略 |
| 跨越恐懼 | 恐懼是舊迴路的保護機制，不是真實的危險信號 |

### 第三階段：增強積極性
| 主題 | 核心概念 |
|---|---|
| 提升心靈韌性 | 韌性不是天生的，是透過小壓力反覆鍛鍊建立的 |
| 培養成長心態 | 固定型 vs 成長型心態的神經基礎 |
| 運動的好處 | 運動釋放 BDNF（腦源性神經營養因子），替大腦施肥 |
| 睡眠的重要性 | 睡眠是神經重組的施工時間，鞏固白天建立的新迴路 |
| 避免過度追求多巴胺 | 多巴胺是「預期」的化學物質，過度刺激讓動力系統失靈 |
| 建立自我信賴 | 每次兌現對自己的承諾，都在強化自我信任的神經迴路 |
