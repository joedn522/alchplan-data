## 為什麼這本書是經典

康納曼是**心理學家拿諾貝爾經濟學獎**的唯一一人（2002）。他和已故夥伴特沃斯基（Amos Tversky）的研究，直接催生了「行為經濟學」這整個領域。

這本書的核心是一個簡單但顛覆的主張：

> **你以為自己是理性的，但你的大腦大部分時間在抄捷徑、犯系統性的錯，而且你完全沒察覺。**

它不是教你「如何聰明」，而是讓你**看見自己思考的盲點**——這些盲點不是因為你笨，是因為人類大腦的設計就是這樣。

---

## 核心架構：系統一 vs 系統二

全書建立在一個比喻上：

| | 系統一（快思） | 系統二（慢想） |
|---|---|---|
| 速度 | 自動、瞬間 | 緩慢、費力 |
| 努力 | 不費力 | 需要專注 |
| 例子 | 2+2=？、看到憤怒的臉 | 17×24=？、停車入庫 |
| 特性 | 直覺、情緒、聯想 | 邏輯、計算、自我懷疑 |
| 問題 | 容易被騙、有系統性偏誤 | 懶惰，常常直接相信系統一 |

**關鍵洞見**：系統二自以為是主角，其實大部分決定都是系統一做的，系統二只是事後幫它找理由。

## 最值得深挖的洞見（待聊）

- **錨點效應**（Ch.11）：一個隨機數字就能改變你的估價
- **可得性偏誤**（Ch.12-13）：你高估了「容易想起來」的事的機率
- **損失趨避**（Ch.28）：失去 100 元的痛 > 得到 100 元的爽，約 2 倍
- **展望理論**（Ch.26）：人對「相對變化」敏感，不是「絕對結果」
- **兩個自我**（Ch.35-38）：記憶中的你和經驗中的你，不是同一個人

## 討論筆記

### 第一部：兩個系統（用工程師角度理解）

**核心對應**：
- **系統一 = 快思** → 像 CPU 的快取／直覺路徑。自動、瞬間、零成本，隨時在跑，關不掉。
- **系統二 = 慢想** → 像主運算／慢路徑。緩慢、費力，要佔用「注意力」這個有限資源。

> 記憶法：系統**一** = 第一反應 = **快**；系統**二** = 要多想一步 = **慢**。
> 書名《**快思**慢想》的順序就是系統一在前、系統二在後。

**一句話總結整個架構**：
> 系統一一直在產生「建議」（直覺、印象、情緒），系統二大部分時候只是無腦 approve 這些建議。問題就出在這個 rubber-stamp。

**系統一在做的事（你沒意識到）**：看到 2+2 直接浮現 4、聽到「麵包和…」自動補「奶油」、看一張臉立刻知道對方在生氣、開車在空路上手腳自動操作、看到中文字無法選擇不讀。
> 它最危險的特性：**永遠在給答案，從不說「我不知道」。**

**系統二在做的事（要用力）**：算 17×24、窄車位停車、報稅、吵雜派對中專心聽一個人講話、檢查複雜邏輯。
> 關鍵特性：**單執行緒、低頻寬**，一次只能做一件事。

**可驗證的實驗**：邊快走邊算 17×24，你會不自覺停下腳步——系統二算力被佔滿，連「維持快走」都分配不到資源。
（康納曼成名研究：人動腦時瞳孔會放大，放大程度精準對應用腦強度——瞳孔 = 系統二的工作負載監視器。）

**球棒與球問題**：球棒+球=1.10 美元，球棒比球貴 1 元，球多少錢？
- 系統一秒答「10 美分」（最省力：把 1.10 拆成 1 和 0.10）
- 正確答案是 **5 美分**（1.05 + 0.05 = 1.10）
- 連哈佛、MIT、普林斯頓學生超過 50% 答錯
- **錯誤根源不是算不出來，是系統二沒被叫醒去檢查**

**惰性的控制者（Ch.3）**：系統二天生遵循「最小努力法則」，能不動腦就不動腦。答錯的人不是不會算，是系統二打瞌睡。

**Asher 的回應**：這個架構「太好理解了」，不需要額外內化。

### Ch.4 聯想機器（The Associative Machine）

系統一的運作原理不是邏輯，是**聯想**，而且是自動、擋不住的。

**聯想活化**：看到「香蕉　嘔吐」兩個詞，零點幾秒內大腦自動：臉微微厭惡、把兩者連在一起、有點反胃、自動編出因果故事（吃香蕉→嘔吐，即使根本沒這回事）。一個念頭像漣漪點亮一整串相關念頭、情緒、身體反應。

**促發效應（priming）**：你剛接觸的東西，會在你毫無察覺下改變接下來的行為。
- 填空 `SO_P`：剛看過 EAT → 填 SOUP；剛看過 WASH → 填 SOAP。同一個空格，前面的字決定答案，而你不知道被影響了。
- **佛羅里達效應**：造句練習藏入「老年」相關字（佛羅里達、健忘、灰白、皺紋，但沒有「慢」或「老」），做完後學生走路明顯變慢。想法直接驅動身體 = **意動效應**。
- 反向也成立：刻意走慢 → 辨認「老年」字變快；點頭聽演講更易同意，搖頭更易反對。

**真正的一拳**：
> 你以為你是自己每個選擇的作者。其實大部分判斷和衝動，都是系統一根據你沒察覺的聯想先幫你決定好的。「我是理性思考才這樣決定」常常是系統二事後編的故事。

**Asher 的精煉理解（已確認精確）**：
> 聯想先污染，行動先發生，系統二事後背書還以為是自己的決定。
> 危險不在系統一愛聯想（那是高效運作方式），而在於這些聯想**偽裝成深思熟慮的結論**，系統二沒看到輸入所以沒機會否決——等它登場，被污染的判斷已變成「你的想法」。
> 例：看履歷時紙質好、排版舒服、名字好唸 → 不自覺覺得這人可信。你以為在評估能力，其實被「順眼」促發了。

### Ch.5 認知放鬆（Cognitive Ease）

大腦隨時在量一個你沒意識到的指標：「我現在處理這件事費不費力？」
- **認知放鬆**：熟悉、好懂、不費力 → 感覺良好、放鬆警戒
- **認知緊張**：陌生、難懂、要用力 → 提高警覺、開始分析

**核心洞見**：
> 凡是讓「處理變輕鬆」的東西，都會讓你覺得它比較真、比較好、比較可信——而你以為你在判斷內容本身。

**四個實驗**：
1. **單純曝光效應**：只是「看過比較多次」就覺得比較順眼、可信，不需理由。廣告與宣傳的引擎：重複→熟悉→相信。演化理由：看過很多次卻沒害死你 = 安全。
2. **真相錯覺**：「讓人相信謊言最可靠的方法就是不斷重複，因為人分不清『熟悉』和『真實』。」
3. **字體實驗**：同一句話，清晰高對比的字 → 覺得比較可信；模糊褪色 → 覺得可疑。連押韻都算（押韻讀起來順 = 放鬆 = 覺得真）。
4. **反轉**：球棒與球印成難讀字體，答對的人反而變多——難讀製造緊張，緊張把系統二叫醒。**讓題目變難讀，反而讓人想得更清楚。**

**心情開關**：心情好 → 系統一掌權 → 直覺、有創意，但更輕信易錯；心情差/警覺 → 系統二上場 → 更分析謹慎，但創意下降。

**對內容/品牌的應用框架**：

| 目標 | 該用 |
|---|---|
| 讓人喜歡、信任、快速接受 | 認知放鬆（順、清楚、重複） |
| 讓人認真思考、不被騙 | 認知緊張（增加一點難度） |

做量化影片時「乾淨圖表 + 簡潔口白 + 重複關鍵結論」不只是美觀，**直接決定觀眾信不信你**。

**Asher 的洞見**：書店那種「一頁一句話 + 浮誇視覺」的書，就是工程化的認知放鬆——大字留白高對比、金句、翻頁快讀得爽，製造「我懂了、好有道理」的感覺。
> 陷阱：那個「好有道理」是認知放鬆製造的錯覺，不是內容有料。讀的時候很爽、讀完什麼都記不住——輕鬆感冒充了「獲得洞見」。**它賣的不是知識，是「我好像變聰明了」的爽感。**

### Ch.6 常態、驚訝與原因

系統一在背景默默做兩件事：

**1. 建立「什麼是正常」的世界模型**：不斷從經驗歸納常態，現實一偏離就亮紅燈 = 驚訝。
- 摩西錯覺：「摩西把每種動物各帶幾隻上方舟？」幾乎沒人發現造方舟的是諾亞——因為摩西也是聖經人物，語境合理，系統一不亮燈直接放行。
- 但「地球每年繞著麻煩轉」你立刻發現不對，因為強烈違反常態。
- 重點：**系統一只抓明顯違規，語境合理的錯誤直接放過。**

**2. 強迫性尋找因果**：
- 物理因果：球撞球，你「看到」因果（其實只是先後）
- 意圖因果：螢幕上幾個三角形移動，人自動描述成「大的欺負小的」「小的在逃跑」——對幾何圖形腦補意圖
- 核心：**我們天生到處找原因和意圖，即使根本沒有。** 這是相信命運、在隨機中看到規律、把巧合當天意的根源。

**Asher 的應用（什麼是常態）**：
> 我覺得自己這個年齡才在這個 level 太低，很不舒服。但那是因為從小在菁英/競爭環境長大，系統一歸納出「這個年齡=該配這個 level」的常態。其實不一定——有些人不以工作為第一要務，待在同樣位置一點都不痛。
> 拆解：讓你不舒服的不是現實，是那條「常態」本身。常態不是真理，是環境餵給你的參考點。痛苦來源不是你的 level，是你拿了一把不一定適合你的尺在量自己。（這正是下一章「錨點效應」的預演，也呼應演員框架：離開＝拒絕一條不服務你的常態，重新選一把尺。）

### Ch.7 驟下結論的機器

**系統一急著下結論，不管證據夠不夠**：「Mindik 聰明、堅強……」你已傾向「她是好領導者」，但下一個字可能是「腐敗又殘忍」。只憑兩個形容詞就跳結論。

**大招 WYSIATI（What You See Is All There Is，你看到的就是全部）**——全書最該記的縮寫：
> 系統一只用眼前現有資訊拼出最連貫的故事，完全不管它沒有的資訊。你的信心來自「故事多連貫」，不是「證據多充分」。

一次解釋一堆偏誤：
- **過度自信**：故事越順越有信心，但「順」≠「對」
- **框架效應**：「95% 瘦肉」vs「5% 肥肉」同一塊肉反應完全不同
- **忽略基本比率**：眼前有生動故事就忘了背後統計

**光環效應**：同一組形容詞順序不同印象天差地別。「聰明、勤奮、衝動、愛批評、固執、善妒」vs 反過來——同一個人，第一種你覺得不錯，第二種覺得討厭。第一印象幫你定調，後面被染色。

**先相信再懷疑**：系統一天生輕信，負責懷疑的系統二很懶常在忙。所以累了、分心時更容易相信假的東西。

**為什麼這兩章對量化工作致命**：
> 跑出漂亮回測 → 系統一腦補因果故事（「這因子有效因為市場心理…」）→ 故事連貫 → 信心爆棚 → 但忽略了沒看到的資料：失敗的策略、樣本外表現、過度擬合、純粹運氣。這是 WYSIATI + 因果腦補的合體。
> 量化的整套紀律（樣本外測試、多重假設檢定校正、警惕 overfitting）本質上就是在對抗系統一這兩個毛病。**回測會給你一個故事，但「故事連貫」不等於「策略有效」。**
